Can you write a palindromic program with a checkio(s) function that checks whether s (a string) is a palindrome?

Using "#" is forbidden.

You can use other methods for the function's definition (for example a lambda). The test tries to run the function "checkio" from your code.

The example of the palindromic code: 

    checkio=lambda x: x#x :x adbmal=oikcehc

But you can not use "#".

Input: A string.

Output: Palindrome or not, a boolean.
